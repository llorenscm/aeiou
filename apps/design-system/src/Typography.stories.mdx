# Typography

Use mixins to change size.

```scss
@use "@aeiou/design-system/src/tools/typography" as mixins;
```

`@include mixins.h400-regular;`

<div className="h400-regular">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</div>

`@include mixins.h300-regular;`

<div className="h300-regular">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</div>
